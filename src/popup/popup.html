AdBlocker Lite

基于 declarativeNetRequest 的广告拦截扩展

版本: 1.0.0

技术栈

  • Manifest V3
  • declarativeNetRequest API
  • Service Worker
  • Content Scripts
  • uBO Lite 规则引擎

规则集 (6 个核心规则集)

  • uBO Filters - 广告 + 追踪 + 恶意软件
  • EasyList - 广告过滤
  • EasyPrivacy - 隐私保护
  • Peter Lowe - 广告/追踪服务器
  • uBO Badware - 恶意网站
  • URLhaus - 恶意 URL

规则通过 scripts/build-rules.mjs 从 uBO Lite 自动生成
